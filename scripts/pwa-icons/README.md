# PWA Icons — upload & wire-up

This folder holds the two PNG icons the PWA manifest references:

- `icon-192.png` — 192×192, used for home-screen, Apple touch icon, notification icon
- `icon-512.png` — 512×512, used for splash / maskable

> Both files are generated by a separate task from
> `D:\cluade\aneh-hashoel\frontend\public\logo.png`. Do **not** edit them by
> hand — regenerate instead.

## Step 1 — Upload to WordPress media library

1. Log in to `https://moreshet-maran.com/wp-admin/`.
2. Go to **Media → Add New**.
3. Upload both files: `icon-192.png` and `icon-512.png`.
4. For each file, open it and **copy the "File URL"** from the right sidebar.

The URLs will look like:

```
https://moreshet-maran.com/wp-content/uploads/YYYY/MM/icon-192.png
https://moreshet-maran.com/wp-content/uploads/YYYY/MM/icon-512.png
```

(`YYYY/MM` is the year/month folder WordPress auto-assigns at upload time.)

## Step 2 — Patch the snippet with the real URLs

Open `..\wp-snippet-pwa.php` and replace the two placeholder URLs (the file
has `2026/04` hard-coded as a placeholder year/month) with the URLs you
copied in Step 1. There are **four places** to update:

1. `aneh_pwa_manifest_data()` — `$icon_192`
2. `aneh_pwa_manifest_data()` — `$icon_512`
3. `wp_head` injection — `$apple_icon`
4. Service worker `push` handler — `icon` / `badge` defaults

Fastest way: find-and-replace on
`https://moreshet-maran.com/wp-content/uploads/2026/04/` with your real
upload path across the file.

## Step 3 — Redeploy the snippet

```bash
python ..\deploy-pwa-snippet.py
```

## Step 4 — Verify

```bash
curl -s https://moreshet-maran.com/wp-json/aneh-pwa/v1/manifest | jq .icons
```

You should see the four icon entries pointing at your real upload URLs,
returning with `Content-Type: application/manifest+json`.

## Alternative — serve icons from this folder

If you'd rather not use the media library, upload the two PNGs anywhere
publicly served over HTTPS and paste those URLs into the snippet the same
way. The only hard requirements are: HTTPS, same origin as the site
(`moreshet-maran.com`) — otherwise the install prompt won't fire — and
matching `Content-Type: image/png`.
